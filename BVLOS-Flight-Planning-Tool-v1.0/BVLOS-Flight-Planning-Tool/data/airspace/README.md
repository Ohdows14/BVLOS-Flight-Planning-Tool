# Airspace Data

Place your airspace boundary files here:

- `canadian_airspace.geojson` (required)
- `*.cub` files (optional)
- `*.sua` files (optional)
- Shapefiles (optional)

## Sources

- NAV Canada aeronautical data
- OpenAIP
- Custom airspace definitions

## Update Frequency

Update every AIRAC cycle (28 days) for current airspace information.
